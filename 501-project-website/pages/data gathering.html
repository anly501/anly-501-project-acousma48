Data Gathering

Data collection is the first and one of the most important steps in all data science research. Data collection is not as simple as you think. If you want to understand data collection, you must first understand what data is.

Not all information can be called data, otherwise information and data would be synonyms. In short, data is information that is meaningful to the study or the activity to be carried out. We get the data we need by formatting the information in the same way.

Why is data collection important? Because it is the premise and guarantee of all subsequent work. Invalid data will affect the quality of the subsequent work, and relatively effective data will naturally make the subsequent work more outstanding. We can understand that a writer needs to collect a lot of materials before he writes a good literary work, or a painter needs to collect inspiration before he finishes a good painting. For a data scientist, collecting good quality data is always one of the most critical parts of the job.

There are many methods for data collection, mainly divided into two types: first, direct acquisition, including survey, interview and online inquiry; The other is indirect access including quotes, social media monitoring and so on. Since this project needs to obtain tweets on Twitter as our data, we adopt the method of Twitter API.

The Twitter API is a developer platform launched by twitter that provides an advanced way to programmatically access public Twitter data. Having code associated with the twitter API in both the R and Python languages allows us to quickly collect public data on twitter.

We used Python and R tools to collect 200 and 1000 new energy tweets, respectively, and collected data on each tweet's favorite count and retweet count.

From the data collected above, it can be intuitively seen that most new energy-related tweets are not sent by popular figures, which shows that ordinary people are also very concerned about new energy. Finally, we can see that the collected data cannot be directly used for the subsequent work, such as the analysis of SVM, clustering and other models, because it contains too much noise, such as invalid words such as amp and some invalid punctuation marks.
