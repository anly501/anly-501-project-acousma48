Naive Bayes

Naive Bayes (NB) in python with Labeled Record Data

The cleaned data was matched by python to the naive bayes model to generate two graphs, the first of confusion metrics and the second of a word cloud excluding irrelevant content.

The first image is confusion metrics, which evaluates the accuracy of a classification, has two dimensions, actual condition and predicted condition. It can easily fraction data. We use 'TP',' FN ','FP' and 'TN' to distinguish different data, which is of great help to our naive Bayes classification.

The second image is a word cloud excluding irrelevant content.When we set the label, we found that a lot of tweets were not related to the topic we wanted to study, and only included the word "new energy". Therefore, we removed these words to generate a new word cloud, from which it can be found that the most relevant theme of tweets related to new energy was "green". "future", "vihicle" and so on.
