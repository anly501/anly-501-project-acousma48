Naive Bayes

Naive Bayes is a classification model that assigns labels to examples of a realistic problem, and labels are limited to classify a realistic problem. Using a simple example from life, for example, if a drink is black, gas-filled and the wrapper on the bottle is red, we may think it is cola. The logic of Naive Bayes is that each characteristic in a label independently affects the probability that the drink is cola.

Naive Bayes classifiers, just as the name suggests, are simple classification models that can be very effective in many cases.

Naive Bayes (NB) in python with Labeled Record Data

In this project, we used simple labels to classify tweets into those related to new energy or those not, and then used Naive Bayes models to match them.

The cleaned data was matched by python to the naive bayes model to generate a graphs, confusion metrics.

The first image is confusion metrics, which evaluates the accuracy of a classification, has two dimensions, actual condition and predicted condition. It can easily fraction data. We use 'TP',' FN ','FP' and 'TN' to distinguish different data, which is of great help to our naive Bayes classification.

We can see from the figure that the performance of NB model is not bad in our data set, but it is not good enough. There are still a lot of data classified as negative.
